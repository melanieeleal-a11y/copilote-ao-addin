# Complément Word — Copilote AO

Hébergement du complément Word interne d'Énergies Locales.

Le contenu applicatif (`taskpane.html`) est **chiffré** (AES-256-GCM) : sans le
mot de passe de déverrouillage, il est inutilisable et illisible. Ce dépôt ne
contient aucune donnée, aucun document et aucun code source en clair.

Installation : voir `installer-complement.ps1` (Windows) ou charger
`manifest.xml` dans Word sur le web (Insertion → Compléments → Charger mon
complément).
